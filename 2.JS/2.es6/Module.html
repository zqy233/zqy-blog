еңЁ ES6 д№ӢеүҚпјҢзӨҫеҢәеҲ¶е®ҡдәҶдёҖдәӣжЁЎеқ—еҠ иҪҪж–№жЎҲпјҢжңҖдё»иҰҒзҡ„жңү CommonJS е’Ң AMD дёӨз§ҚгҖӮеүҚиҖ…з”ЁдәҺжңҚеҠЎеҷЁпјҢеҗҺиҖ…з”ЁдәҺжөҸи§ҲеҷЁгҖӮES6 еңЁиҜӯиЁҖж ҮеҮҶзҡ„еұӮйқўдёҠпјҢе®һзҺ°дәҶжЁЎеқ—еҠҹиғҪпјҢиҖҢдё”е®һзҺ°еҫ—зӣёеҪ“з®ҖеҚ•пјҢе®Ңе…ЁеҸҜд»ҘеҸ–д»Ј CommonJS е’Ң AMD и§„иҢғпјҢжҲҗдёәжөҸи§ҲеҷЁе’ҢжңҚеҠЎеҷЁйҖҡз”Ёзҡ„жЁЎеқ—и§ЈеҶіж–№жЎҲгҖӮ

ES6 жЁЎеқ—зҡ„и®ҫи®ЎжҖқжғіжҳҜе°ҪйҮҸзҡ„йқҷжҖҒеҢ–пјҢдҪҝеҫ—зј–иҜ‘ж—¶е°ұиғҪзЎ®е®ҡжЁЎеқ—зҡ„дҫқиө–е…ізі»пјҢд»ҘеҸҠиҫ“е…Ҙе’Ңиҫ“еҮәзҡ„еҸҳйҮҸгҖӮCommonJS е’Ң AMD жЁЎеқ—пјҢйғҪеҸӘиғҪеңЁиҝҗиЎҢж—¶зЎ®е®ҡиҝҷдәӣдёңиҘҝгҖӮжҜ”еҰӮпјҢCommonJS жЁЎеқ—е°ұжҳҜеҜ№иұЎпјҢиҫ“е…Ҙж—¶еҝ…йЎ»жҹҘжүҫеҜ№иұЎеұһжҖ§гҖӮ

йҷӨдәҶйқҷжҖҒеҠ иҪҪеёҰжқҘзҡ„еҗ„з§ҚеҘҪеӨ„пјҢES6 жЁЎеқ—иҝҳжңүд»ҘдёӢеҘҪеӨ„гҖӮ

  • дёҚеҶҚйңҖиҰҒUMDжЁЎеқ—ж јејҸдәҶпјҢе°ҶжқҘжңҚеҠЎеҷЁе’ҢжөҸи§ҲеҷЁйғҪдјҡж”ҜжҢҒ ES6 жЁЎеқ—ж јејҸгҖӮзӣ®еүҚпјҢйҖҡиҝҮеҗ„з§Қе·Ҙе…·еә“пјҢе…¶е®һе·Із»ҸеҒҡеҲ°дәҶиҝҷдёҖзӮ№гҖӮ
  • е°ҶжқҘжөҸи§ҲеҷЁзҡ„ж–° API е°ұиғҪз”ЁжЁЎеқ—ж јејҸжҸҗдҫӣпјҢдёҚеҶҚеҝ…йЎ»еҒҡжҲҗе…ЁеұҖеҸҳйҮҸжҲ–иҖ…navigatorеҜ№иұЎзҡ„еұһжҖ§гҖӮ
  • дёҚеҶҚйңҖиҰҒеҜ№иұЎдҪңдёәе‘ҪеҗҚз©әй—ҙпјҲжҜ”еҰӮMathеҜ№иұЎпјүпјҢжңӘжқҘиҝҷдәӣеҠҹиғҪеҸҜд»ҘйҖҡиҝҮжЁЎеқ—жҸҗдҫӣгҖӮ
