Intersection Observer API #

Intersection Observer API жҸҗдҫӣдәҶдёҖз§ҚејӮжӯҘжЈҖжөӢзӣ®ж Үе…ғзҙ дёҺзҘ–е…Ҳе…ғзҙ жҲ– viewport зӣёдәӨжғ…еҶөеҸҳеҢ–зҡ„ж–№жі•гҖӮ

иҝҮеҺ»пјҢиҰҒжЈҖжөӢдёҖдёӘе…ғзҙ жҳҜеҗҰеҸҜи§ҒжҲ–иҖ…дёӨдёӘе…ғзҙ жҳҜеҗҰзӣёдәӨе№¶дёҚе®№жҳ“пјҢеҫҲеӨҡи§ЈеҶіеҠһжі•дёҚеҸҜйқ жҲ–жҖ§иғҪеҫҲе·®гҖӮз„¶иҖҢпјҢйҡҸзқҖдә’иҒ”зҪ‘зҡ„еҸ‘еұ•пјҢиҝҷз§ҚйңҖжұӮеҚҙдёҺж—ҘдҝұеўһпјҢжҜ”еҰӮпјҢдёӢйқўиҝҷдәӣжғ…еҶөйғҪйңҖиҰҒз”ЁеҲ°зӣёдәӨжЈҖжөӢпјҡ

  • еӣҫзүҮжҮ’еҠ иҪҪвҖ”вҖ”еҪ“еӣҫзүҮж»ҡеҠЁеҲ°еҸҜи§Ғж—¶жүҚиҝӣиЎҢеҠ иҪҪ
  • еҶ…е®№ж— йҷҗж»ҡеҠЁвҖ”вҖ”д№ҹе°ұжҳҜз”ЁжҲ·ж»ҡеҠЁеҲ°жҺҘиҝ‘еҶ…е®№еә•йғЁж—¶зӣҙжҺҘеҠ иҪҪжӣҙеӨҡпјҢиҖҢж— йңҖз”ЁжҲ·ж“ҚдҪңзҝ»йЎөпјҢз»ҷз”ЁжҲ·дёҖз§ҚзҪ‘йЎөеҸҜд»Ҙж— йҷҗж»ҡеҠЁзҡ„й”ҷи§ү
  • жЈҖжөӢе№ҝе‘Ҡзҡ„жӣқе…үжғ…еҶөвҖ”вҖ”дёәдәҶи®Ўз®—е№ҝе‘Ҡж”¶зӣҠпјҢйңҖиҰҒзҹҘйҒ“е№ҝе‘Ҡе…ғзҙ зҡ„жӣқе…үжғ…еҶө
  • еңЁз”ЁжҲ·зңӢи§ҒжҹҗдёӘеҢәеҹҹж—¶жү§иЎҢд»»еҠЎжҲ–ж’ӯж”ҫеҠЁз”»

иҝҮеҺ»пјҢзӣёдәӨжЈҖжөӢйҖҡеёёиҰҒз”ЁеҲ°дәӢд»¶зӣ‘еҗ¬пјҢе№¶дё”йңҖиҰҒйў‘з№Ғи°ғз”Ё Element.getBoundingClientRect() ж–№жі•д»ҘиҺ·еҸ–зӣёе…іе…ғзҙ зҡ„иҫ№з•ҢдҝЎжҒҜгҖӮдәӢд»¶зӣ‘еҗ¬е’Ңи°ғз”Ё Element.getBoundingClientRect() йғҪжҳҜеңЁдё»зәҝзЁӢдёҠиҝҗиЎҢпјҢеӣ жӯӨйў‘з№Ғи§ҰеҸ‘гҖҒи°ғз”ЁеҸҜиғҪдјҡйҖ жҲҗжҖ§иғҪй—®йўҳгҖӮиҝҷз§ҚжЈҖжөӢж–№жі•жһҒе…¶жҖӘејӮдё”дёҚдјҳйӣ…гҖӮ

еҒҮеҰӮжңүдёҖдёӘж— йҷҗж»ҡеҠЁзҡ„зҪ‘йЎөпјҢејҖеҸ‘иҖ…дҪҝз”ЁдәҶдёҖдёӘз¬¬дёүж–№еә“жқҘз®ЎзҗҶж•ҙдёӘйЎөйқўзҡ„е№ҝе‘ҠпјҢеҸҲз”ЁдәҶеҸҰеӨ–дёҖдёӘеә“жқҘе®һзҺ°ж¶ҲжҒҜзӣ’еӯҗе’ҢзӮ№иөһпјҢе№¶дё”йЎөйқўжңүеҫҲеӨҡеҠЁз”»пјҲиҜ‘жіЁпјҡеҠЁз”»еҫҖеҫҖж„Ҹе‘ізқҖиҫғй«ҳзҡ„жҖ§иғҪж¶ҲиҖ—пјүгҖӮдёӨдёӘеә“йғҪжңүиҮӘе·ұзҡ„зӣёдәӨжЈҖжөӢзЁӢеәҸпјҢйғҪиҝҗиЎҢеңЁдё»зәҝзЁӢйҮҢпјҢиҖҢзҪ‘з«ҷзҡ„ејҖеҸ‘иҖ…еҜ№иҝҷдәӣеә“зҡ„еҶ…йғЁе®һзҺ°зҹҘд№Ӣз”ҡе°‘пјҢжүҖд»Ҙе№¶жңӘж„ҸиҜҶеҲ°жңүд»Җд№Ҳй—®йўҳгҖӮдҪҶеҪ“з”ЁжҲ·ж»ҡеҠЁйЎөйқўж—¶пјҢиҝҷдәӣзӣёдәӨжЈҖжөӢзЁӢеәҸе°ұдјҡеңЁйЎөйқўж»ҡеҠЁеӣһи°ғеҮҪж•°йҮҢдёҚеҒңи§ҰеҸ‘и°ғз”ЁпјҢйҖ жҲҗжҖ§иғҪй—®йўҳпјҢдҪ“йӘҢж•Ҳжһңи®©дәәеӨұжңӣгҖӮ

Intersection Observer API дјҡжіЁеҶҢдёҖдёӘеӣһи°ғеҮҪж•°пјҢжҜҸеҪ“иў«зӣ‘и§Ҷзҡ„е…ғзҙ иҝӣе…ҘжҲ–иҖ…йҖҖеҮәеҸҰеӨ–дёҖдёӘе…ғзҙ ж—¶ (жҲ–иҖ… viewport )пјҢжҲ–иҖ…дёӨдёӘе…ғзҙ зҡ„зӣёдәӨйғЁеҲҶеӨ§е°ҸеҸ‘з”ҹеҸҳеҢ–ж—¶пјҢиҜҘеӣһи°ғж–№жі•дјҡиў«и§ҰеҸ‘жү§иЎҢгҖӮиҝҷж ·пјҢжҲ‘д»¬зҪ‘з«ҷзҡ„дё»зәҝзЁӢдёҚйңҖиҰҒеҶҚдёәдәҶзӣ‘еҗ¬е…ғзҙ зӣёдәӨиҖҢиҫӣиӢҰеҠідҪңпјҢжөҸи§ҲеҷЁдјҡиҮӘиЎҢдјҳеҢ–е…ғзҙ зӣёдәӨз®ЎзҗҶгҖӮ

жіЁж„Ҹ Intersection Observer API ж— жі•жҸҗдҫӣйҮҚеҸ зҡ„еғҸзҙ дёӘж•°жҲ–иҖ…е…·дҪ“е“ӘдёӘеғҸзҙ йҮҚеҸ пјҢд»–зҡ„жӣҙеёёи§Ғзҡ„дҪҝз”Ёж–№ејҸжҳҜвҖ”вҖ”еҪ“дёӨдёӘе…ғзҙ зӣёдәӨжҜ”дҫӢеңЁ N% е·ҰеҸіж—¶пјҢи§ҰеҸ‘еӣһи°ғпјҢд»Ҙжү§иЎҢжҹҗдәӣйҖ»иҫ‘гҖӮ

Intersection observer зҡ„жҰӮеҝөе’Ңз”Ёжі• #

Intersection Observer API е…Ғи®ёдҪ й…ҚзҪ®дёҖдёӘеӣһи°ғеҮҪж•°пјҢеҪ“д»ҘдёӢжғ…еҶөеҸ‘з”ҹж—¶дјҡиў«и°ғз”Ё

  • жҜҸеҪ“зӣ®ж Ү (target) е…ғзҙ дёҺи®ҫеӨҮи§ҶзӘ—жҲ–иҖ…е…¶д»–жҢҮе®ҡе…ғзҙ еҸ‘з”ҹдәӨйӣҶзҡ„ж—¶еҖҷжү§иЎҢгҖӮи®ҫеӨҮи§ҶзӘ—жҲ–иҖ…е…¶д»–е…ғзҙ жҲ‘д»¬з§°е®ғдёәж №е…ғзҙ жҲ–ж № (root)гҖӮ
  • Observer з¬¬дёҖж¬Ўзӣ‘еҗ¬зӣ®ж Үе…ғзҙ зҡ„ж—¶еҖҷ

йҖҡеёёпјҢжӮЁйңҖиҰҒе…іжіЁж–ҮжЎЈжңҖжҺҘиҝ‘зҡ„еҸҜж»ҡеҠЁзҘ–е…Ҳе…ғзҙ зҡ„дәӨйӣҶжӣҙж”№пјҢеҰӮжһңе…ғзҙ дёҚжҳҜеҸҜж»ҡеҠЁе…ғзҙ зҡ„еҗҺд»ЈпјҢеҲҷй»ҳи®Өдёәи®ҫеӨҮи§ҶзӘ—гҖӮеҰӮжһңиҰҒи§ӮеҜҹзӣёеҜ№дәҺж № (root) е…ғзҙ зҡ„дәӨйӣҶпјҢиҜ·жҢҮе®ҡж № (root) е…ғзҙ дёәnullгҖӮ

ж— и®әжӮЁжҳҜдҪҝз”Ёи§ҶеҸЈиҝҳжҳҜе…¶д»–е…ғзҙ дҪңдёәж №пјҢAPI йғҪд»ҘзӣёеҗҢзҡ„ж–№ејҸе·ҘдҪңпјҢеҸӘиҰҒзӣ®ж Үе…ғзҙ зҡ„еҸҜи§ҒжҖ§еҸ‘з”ҹеҸҳеҢ–пјҢе°ұдјҡжү§иЎҢжӮЁжҸҗдҫӣзҡ„еӣһи°ғеҮҪж•°пјҢд»Ҙдҫҝе®ғдёҺжүҖйңҖзҡ„дәӨеҸүзӮ№дәӨеҸүгҖӮ

зӣ®ж Ү (target) е…ғзҙ дёҺж № (root) е…ғзҙ д№Ӣй—ҙзҡ„дәӨеҸүеәҰжҳҜдәӨеҸүжҜ” (intersection ratio)гҖӮиҝҷжҳҜзӣ®ж Ү (target) е…ғзҙ зӣёеҜ№дәҺж № (root) зҡ„дәӨйӣҶзҷҫеҲҶжҜ”зҡ„иЎЁзӨәпјҢе®ғзҡ„еҸ–еҖјеңЁ 0.0 е’Ң 1.0 д№Ӣй—ҙгҖӮ

еҲӣе»әдёҖдёӘ intersection observer #

еҲӣе»әдёҖдёӘ IntersectionObserver еҜ№иұЎпјҢе№¶дј е…Ҙзӣёеә”еҸӮж•°е’Ңеӣһи°ғз”ЁеҮҪж•°пјҢиҜҘеӣһи°ғеҮҪж•°е°ҶдјҡеңЁзӣ®ж Ү (target) е…ғзҙ е’Ңж № (root) е…ғзҙ зҡ„дәӨйӣҶеӨ§е°Ҹи¶…иҝҮйҳҲеҖј (threshold) и§„е®ҡзҡ„еӨ§е°Ҹж—¶еҖҷиў«жү§иЎҢгҖӮ

let options = {
  root: document.querySelector('#scrollArea'),
  rootMargin: '0px',
  threshold: 1.0

let observer = new IntersectionObserver(callback, options);

Intersection observer options #

дј йҖ’еҲ° IntersectionObserver() жһ„йҖ еҮҪж•°зҡ„ options еҜ№иұЎпјҢе…Ғи®ёжӮЁжҺ§еҲ¶и§ӮеҜҹиҖ…зҡ„еӣһи°ғеҮҪж•°зҡ„иў«и°ғз”Ёж—¶зҡ„зҺҜеўғгҖӮе®ғжңүд»ҘдёӢеӯ—ж®өпјҡ

  • root

    жҢҮе®ҡж № (root) е…ғзҙ пјҢз”ЁдәҺжЈҖжҹҘзӣ®ж Үзҡ„еҸҜи§ҒжҖ§гҖӮеҝ…йЎ»жҳҜзӣ®ж Үе…ғзҙ зҡ„зҲ¶зә§е…ғзҙ гҖӮеҰӮжһңжңӘжҢҮе®ҡжҲ–иҖ…дёәnullпјҢеҲҷй»ҳи®ӨдёәжөҸи§ҲеҷЁи§ҶзӘ—гҖӮ

  • rootMargin

    ж № (root) е…ғзҙ зҡ„еӨ–иҫ№и·қгҖӮзұ»дјјдәҺ CSS дёӯзҡ„ margin еұһжҖ§пјҢжҜ”еҰӮ "10px 20px 30px 40px" (topгҖҒrightгҖҒbottomгҖҒleft)гҖӮеҰӮжһңжңүжҢҮе®ҡ root еҸӮж•°пјҢеҲҷ rootMargin д№ҹеҸҜд»ҘдҪҝз”ЁзҷҫеҲҶжҜ”жқҘеҸ–еҖјгҖӮиҜҘеұһжҖ§еҖјжҳҜз”ЁдҪң root е…ғзҙ е’Ң target еҸ‘з”ҹдәӨйӣҶж—¶еҖҷзҡ„и®Ўз®—дәӨйӣҶзҡ„еҢәеҹҹиҢғеӣҙпјҢдҪҝз”ЁиҜҘеұһжҖ§еҸҜд»ҘжҺ§еҲ¶ root е…ғзҙ жҜҸдёҖиҫ№зҡ„ж”¶зј©жҲ–иҖ…жү©еј гҖӮй»ҳи®ӨеҖјдёәеӣӣдёӘиҫ№и·қе…ЁжҳҜ 0гҖӮ

  • threshold

    еҸҜд»ҘжҳҜеҚ•дёҖзҡ„ number д№ҹеҸҜд»ҘжҳҜ number ж•°з»„пјҢtarget е…ғзҙ е’Ң root е…ғзҙ зӣёдәӨзЁӢеәҰиҫҫеҲ°иҜҘеҖјзҡ„ж—¶еҖҷ IntersectionObserver жіЁеҶҢзҡ„еӣһи°ғеҮҪж•°е°Ҷдјҡиў«жү§иЎҢгҖӮеҰӮжһңдҪ еҸӘжҳҜжғіиҰҒжҺўжөӢеҪ“ target е…ғзҙ зҡ„еңЁ root е…ғзҙ дёӯзҡ„еҸҜи§ҒжҖ§и¶…иҝҮ 50% зҡ„ж—¶еҖҷпјҢдҪ еҸҜд»ҘжҢҮе®ҡиҜҘеұһжҖ§еҖјдёә 0.5гҖӮеҰӮжһңдҪ жғіиҰҒ target е…ғзҙ еңЁ root е…ғзҙ зҡ„еҸҜи§ҒзЁӢеәҰжҜҸеӨҡ 25% е°ұжү§иЎҢдёҖж¬Ўеӣһи°ғпјҢйӮЈд№ҲдҪ еҸҜд»ҘжҢҮе®ҡдёҖдёӘж•°з»„ [0, 0.25, 0.5, 0.75, 1]гҖӮй»ҳи®ӨеҖјжҳҜ 0 (ж„Ҹе‘ізқҖеҸӘиҰҒжңүдёҖдёӘ target еғҸзҙ еҮәзҺ°еңЁ root е…ғзҙ дёӯпјҢеӣһи°ғеҮҪж•°е°Ҷдјҡиў«жү§иЎҢ)гҖӮиҜҘеҖјдёә 1.0 еҗ«д№үжҳҜеҪ“ target е®Ңе…ЁеҮәзҺ°еңЁ root е…ғзҙ дёӯж—¶еҖҷ еӣһи°ғжүҚдјҡиў«жү§иЎҢгҖӮ
