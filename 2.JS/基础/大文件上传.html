йңҖжұӮеҲҶжһҗ #

й’ҲеҜ№еӨ§ж–Үд»¶дёҠдј пјҢжңҖе°‘еҒҡеҲ°д»ҘдёӢеҮ зӮ№

  • еӨ§ж–Үд»¶еҲҮеүІпјҢеҲҶзүҮдёҠдј 
  • еҰӮжһңжңүйғЁеҲҶеҲҮзүҮдёҠдј еӨұиҙҘдәҶпјҢжҲ‘д»¬еёҢжңӣжҸҗйҶ’з”ЁжҲ·йҮҚж–°дёҠдј пјҢе№¶дё”дёҠдј жҲҗеҠҹдёҚйңҖиҰҒдёҠдј 
  • жңҖеҘҪиғҪжңүдёҠдј зҡ„иҝӣеәҰжҸҗзӨә

д»Җд№ҲжҳҜ web worker #

Web Worker зҡ„дҪңз”ЁпјҢе°ұжҳҜдёә JavaScript еҲӣйҖ еӨҡзәҝзЁӢзҺҜеўғпјҢе…Ғи®ёдё»зәҝзЁӢеҲӣе»ә Worker зәҝзЁӢпјҢе°ҶдёҖдәӣд»»еҠЎеҲҶй…Қз»ҷеҗҺиҖ…иҝҗиЎҢгҖӮеңЁдё»зәҝзЁӢиҝҗиЎҢзҡ„еҗҢж—¶пјҢWorker зәҝзЁӢеңЁеҗҺеҸ°иҝҗиЎҢпјҢдёӨиҖ…дә’дёҚе№Іжү°гҖӮзӯүеҲ° Worker зәҝзЁӢе®ҢжҲҗи®Ўз®—д»»еҠЎпјҢеҶҚжҠҠз»“жһңиҝ”еӣһз»ҷдё»зәҝзЁӢгҖӮиҝҷж ·зҡ„еҘҪеӨ„жҳҜпјҢдёҖдәӣи®Ўз®—еҜҶйӣҶеһӢжҲ–й«ҳе»¶иҝҹзҡ„д»»еҠЎпјҢиў« Worker зәҝзЁӢиҙҹжӢ…дәҶпјҢдё»зәҝзЁӢпјҲйҖҡеёёиҙҹиҙЈ UI дәӨдә’пјүе°ұдјҡеҫҲжөҒз•…пјҢдёҚдјҡиў«йҳ»еЎһжҲ–жӢ–ж…ўгҖӮ

еӨ§ж–Үд»¶дёҠдј зҡ„еҹәжң¬еҠҹиғҪе°ұе·®дёҚеӨҡе®ҢжҲҗдәҶгҖӮ

ж–Үд»¶з§’дј  #

иҝҷдёӘеҠҹиғҪзҡ„ж„ҸжҖқе°ұжҳҜиҜҙпјҢжҲ‘д»¬еңЁж–Үд»¶дёҠдј д№ӢеүҚпјҢеҺ»й—®дёҖдёӢжңҚеҠЎеҷЁпјҢдҪ жңүжІЎжңүиҝҷдёӘж–Үд»¶е‘ҖпјҢдҪ жІЎжңүзҡ„иҜқжҲ‘е°ұејҖе§ӢдёҠдј пјҢдҪ иҰҒжҳҜжңүзҡ„иҜқжҲ‘е°ұеҒ·дёӘжҮ’пјҢз”ЁдҪ жңүзҡ„жҲ‘е°ұдёҚдёҠдј дәҶ

жүҖд»ҘйңҖиҰҒе®һзҺ°дёҖдёӘжЈҖжөӢжҺҘеҸЈ(verify)пјҢеҺ»иҜўй—®жңҚеҠЎеҷЁжңүжІЎжңүиҝҷдёӘж–Үд»¶пјҢеӣ дёәжҲ‘д»¬д№ӢеүҚжҳҜи®Ўз®—иҝҮж–Үд»¶зҡ„ hashзҡ„пјҢиғҪдҝқиҜҒж–Үд»¶зҡ„е”ҜдёҖжҖ§гҖӮе°ұз”ЁиҝҷдёӘhashе°ұиғҪе”ҜдёҖзҡ„еҲӨж–ӯиҝҷдёӘж–Үд»¶гҖӮжүҖд»ҘиҝҷдёӘжҺҘеҸЈзҡ„жҖқи·Ҝд№ҹеҫҲз®ҖеҚ•пјҢе°ұжҳҜеҲӨж–ӯжҲ‘д»¬зҡ„ targetзӣ®еҪ•дёӢжҳҜеҗҰеӯҳеңЁиҝҷдёӘж–Үд»¶

ж–ӯзӮ№з»ӯдј  #

ж–ӯзӮ№з»ӯдј зҡ„ж„ҸжҖқе°ұжҳҜжҲ‘д»¬дёҠдј зҡ„ж—¶еҖҷеҰӮжһңж–Үд»¶дёҠдј еӨұиҙҘдәҶпјҢжҲ‘д»¬д№ӢеҗҺеңЁдёҠдј дёҖж¬Ўзҡ„ж—¶еҖҷпјҢеҸӘдёҠдј жҲ‘д»¬д№ӢеүҚеӨұиҙҘзҡ„ж–Үд»¶пјҢжҲҗеҠҹзҡ„ж–Үд»¶жҲ‘д»¬е°ұи·іиҝҮ

ж–Үд»¶еҲҮзүҮ #

йҖүжӢ©ж–Үд»¶жҳҜдҪҝз”Ёзҡ„inputиҫ“е…ҘжЎҶпјҢиҺ·еҸ–еҲ°йҖүжӢ©зҡ„еҖјд№ҹеҫҲз®ҖеҚ•гҖӮж–Үд»¶еҲҮзүҮзҡ„ж ёеҝғе°ұжҳҜж–Үд»¶еҜ№иұЎзҡ„slice ж–№жі•пјҢзұ»дјјж•°з»„пјҢжҲ‘д»¬еҸҜд»Ҙи°ғз”ЁиҝҷдёӘж–№жі•иҺ·еҸ–еҲ°ж–Үд»¶зҡ„жҹҗдёҖж®ө

ж–Үд»¶е”ҜдёҖеҖј #

еҰӮдҪ•е‘ҠиҜүеҗҺз«ҜпјҢдёҠдј зҡ„дёӨдёӘж–Үд»¶жҳҜдёҚжҳҜеҗҢдёҖдёӘж–Үд»¶пјҢжҳҫз„¶пјҢеҰӮжһңдҪҝз”Ёж–Үд»¶еҗҚдҪңдёәе”ҜдёҖж ҮиҜҶиӮҜе®ҡдёҚеӨӘеҘҪгҖӮиҝҷдёӘж—¶еҖҷжҲ‘д»¬жғіеҲ°еҸҜд»ҘдҪҝз”Ёmd5еҜ№ж–Үд»¶еҠ еҜҶиҺ·еҸ–е”ҜдёҖзҡ„hashеҖј

з”ҹжҲҗhashеҖјзҡ„ж–№жі•жҲ‘д»¬жҳҜи°ғз”Ё spark-md5 иҝҷдёӘеә“пјҢеңЁи®Ўз®—hashзҡ„ж—¶еҖҷжҳҜйқһеёёж¶ҲиҖ—и®Ўз®—жңәзҡ„CPUзҡ„дјҡйҖ жҲҗжөҸи§ҲеҷЁзҡ„еҚЎйЎҝпјҢдёәдәҶдјҳеҢ–дҪ“йӘҢжҲ‘д»¬дҪҝз”Ё web-worker еңЁ worker зәҝзЁӢи®Ўз®— hash

еңЁ worker зәҝзЁӢдёӯпјҢжҺҘеҸ—ж–Үд»¶еҲҮзүҮ fileChunkListпјҢеҲ©з”Ё FileReader иҜ»еҸ–жҜҸдёӘеҲҮзүҮзҡ„ ArrayBuffer е№¶дёҚж–ӯдј е…Ҙ spark-md5 дёӯпјҢжҜҸи®Ўз®—е®ҢдёҖдёӘеҲҮзүҮйҖҡиҝҮ postMessage еҗ‘дё»зәҝзЁӢеҸ‘йҖҒдёҖдёӘиҝӣеәҰдәӢд»¶пјҢе…ЁйғЁе®ҢжҲҗеҗҺе°ҶжңҖз»Ҳзҡ„ hash еҸ‘йҖҒз»ҷдё»зәҝзЁӢ

жҡӮеҒңдёҠдј  #

жҲ‘д»¬е…ҲиҮӘе·ұжүӢеҠЁе®һзҺ°дёҖдёӘжҢүй’®пјҢзӮ№еҮ»д№ӢеҗҺе°ұеҒңжӯўеҪ“еүҚзҡ„дёҠдј жғ…еҶөгҖӮжЁЎжӢҹдәҶдёҠдј еӨұиҙҘ

иҝҷдёӘжҖқи·ҜиӮҜе®ҡе°ұжҳҜиҰҒж”№иЈ…жҲ‘д»¬зҡ„ requestж–№жі•пјҢеңЁж”№иЈ…д№ӢеүҚжҲ‘д»¬йңҖиҰҒзҹҘйҒ“ XMLHttpRequestеҜ№иұЎжҳҜеҸҜд»ҘиҮӘе·ұдё»еҠЁеҒңжӯўеҪ“еүҚзҡ„зҪ‘з»ңиҝһжҺҘзҡ„пјҢдёҚзҹҘйҒ“зҡ„еҗҢеӯҰиЎҘиЎҘиҜҫгҖӮ

иҝҷж ·жҲ‘д»¬еҸӘйңҖиҰҒдҪҝз”ЁдёҖдёӘе…¬е…ұзҡ„ж•°з»„пјҢжҜҸдёҖж¬ЎеҸ‘иҜ·жұӮзҡ„ж—¶еҖҷйғҪдҝқеӯҳжҲ‘д»¬еҪ“еүҚзҡ„иҝҷдёӘXMLHttpRequestеҜ№иұЎпјҢеҪ“иҜ·жұӮжҲҗеҠҹд№ӢеҗҺпјҢжҲ‘д»¬е°ұз§»йҷӨиҝҷдёӘеҜ№иұЎпјҢеҪ“зӮ№еҮ»жҡӮеҒңжҢүй’®зҡ„ж—¶еҖҷжҲ‘д»¬е°ұйҒҚеҺҶиҝҷдёӘж•°з»„и°ғз”ЁжҜҸдёҖдёӘXMLHttpRequestзҡ„abrotж–№жі•е°ұеҸҜд»ҘеҸ–ж¶ҲдёҠдј дәҶгҖӮ
