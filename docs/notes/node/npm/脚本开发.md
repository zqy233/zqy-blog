# 开发脚本

1.创建空项目

```bash
mkdir git-quick-push
cd git-quick-push
npm init -y
```

2.新建index.js

行首加入 #!/usr/bin/env node 指定当前脚本由node.js进行解析

```js
#! /usr/bin/env node
console.log('hello world')
```

3.配置package.json

> node.js 内置了对命令行操作的支持，package.json 中的 bin 字段可以定义命令名和关联的执行文件

```json
  "bin": {
    "git-push": "index.js"
  }
```

4.执行npm link链接命令到全局

```bash
npm link
```

现在可以本地使用bin里的命令

```bash
git-push
hello world
```
