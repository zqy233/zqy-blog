## template

存放模板文件，包含基础模板和增量模板

create-vue的核心思路是，根据用户选择项，把基础模板和增量模板拼接成新的模板，这样所有模板的总体积更小，包生成模板速度也更快

### .gitmodules

同时，所有最终生成的模板存放在另一个仓库

```
[submodule "playground"]
	path = playground
	url = https://github.com/vuejs/create-vue-templates.git
```
