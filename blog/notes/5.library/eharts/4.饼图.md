# 饼图

> 所有实例代码均可在ehcarts的[background](https://echarts.apache.org/examples/zh/editor.html)环境中直接运行

## 颜色设置

```json
option  = {
  color: ["#6B9CF5", "#8FD59F", "#CA8EE3"],
  series: [
    {
      type: "pie",
      radius: ["40%", "60%"],
      data: [
        { value: 15.79, name: "故障" },
        { value: 10.53, name: "正常" },
        { value: 47.37, name: "其他" }
      ]
    }
  ]
}
```

## 饼图的左右位置

```js
series: [
  {
    ...
    center: ['30%', '50%'],
  }
]
```

## 标题居中

```js
left:"center"
```

## 悬浮文本

```json
option  = {
   tooltip: {
    trigger: "item"
  },
  series: [
    {
      type: "pie",
      radius: ["40%", "60%"],
      data: [
        { value: 15.79, name: "故障" },
        { value: 10.53, name: "正常" },
        { value: 47.37, name: "其他" }
      ]
    }
  ]
}
```

## 文本标签显示百分比

```json
option = {
  series: [
    {
      type: "pie",
      radius: ["40%", "60%"],
      label: {
        show: true,
        position: "outside",
        formatter: ["{a|{d}}%", "{b|{b}}"].join("\n"),
        rich: {
          a: {
            fontWeight: "bold",
            lineHeight: 30,
            fontSize: 20
          },
          b: {
            fontSize: 14
          }
        }
      },
      data: [
        { value: 15.79, name: "故障" },
        { value: 10.53, name: "正常" },
        { value: 47.37, name: "其他" }
      ]
    }
  ]
}
```
