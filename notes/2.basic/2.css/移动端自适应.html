px/em/rem #

px: еғҸзҙ пјҢе®ғжҳҜдёҖдёӘеӣәе®ҡеӨ§е°Ҹзҡ„еҚ•е…ғпјҢеғҸзҙ зҡ„и®Ўз®—жҳҜй’ҲеҜ№еұҸе№•зҡ„пјҢдёҖдёӘеғҸзҙ пјҲ1px)е°ұжҳҜеұҸе№•зҡ„дёҖдёӘзӮ№пјҢдҪҶжҳҜеӣ дёәе®ғжҳҜеӣәе®ҡеӨ§е°Ҹзҡ„пјҢжүҖд»ҘдёҚиғҪеӨҹиҮӘйҖӮеә”

em: emжҳҜзӣёеҜ№дәҺзҲ¶зә§е…ғзҙ зҡ„еҚ•дҪҚпјҢе®ғжҳҜз”ЁжқҘи®ҫзҪ®ж–Үжң¬зҡ„еӯ—дҪ“е°әеҜёпјҢзӣёеҜ№дәҺеҪ“еүҚеҜ№иұЎеҶ…зҡ„ж–Үжң¬зҡ„еӯ—дҪ“е°әеҜёпјӣдёҖиҲ¬жөҸи§ҲеҷЁй»ҳи®Ө(1em=16px)пјҢдјҡйҡҸзҲ¶зә§е…ғзҙ зҡ„еұһжҖ§пјҲfont-sizeжҲ–е…¶д»–еұһжҖ§пјүеҸҳеҢ–иҖҢеҸҳеҢ–

rem: (css3ж–°еўһзҡ„зӣёеҜ№й•ҝеәҰеҚ•дҪҚ)пјҢзӣёеҜ№дәҺж №зӣ®еҪ•пјҢеҚіHTMLе…ғзҙ пјҢжүҖд»ҘеҸӘиҰҒеңЁHTMLж ҮзӯҫдёҠи®ҫзҪ®еӯ—дҪ“еӨ§е°ҸпјҲfont-size)пјҢж–ҮжЎЈдёӯзҡ„еӯ—дҪ“еӨ§е°ҸйғҪдјҡд»ҘжӯӨдёәеҸӮз…§ж ҮеҮҶпјҢдёҖиҲ¬з”ЁдәҺиҮӘйҖӮеә”еёғеұҖ

remдёҺfont-sizeжҚўз®— #

1rem=htmlж Үзӯҫзҡ„font-sizeеӨ§е°Ҹ

жөҸи§ҲеҷЁдёҖиҲ¬й»ҳи®Өеӯ—еҸ·дёә16pxпјҢеҚіfont-size: 16pxпјҢеҲҷ1remдёә16px

css
html {
  font-size: 100% // зӣёеҪ“дәҺ16px
}
div {
  width: 3rem  // зӣёеҪ“дәҺ3*16px = 48px
}

  font-size: 200% // зӣёеҪ“дәҺ32px
}

  width: 3rem  // зӣёеҪ“дәҺ3*32px = 96px
}
